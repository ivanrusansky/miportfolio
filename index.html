Acerca de mí

Soy Iván Rusansky, Desarrollador Front-End especializado en la creación de sitios web modernos, funcionales y adaptados a las necesidades de cada cliente. Inicié mi formación en 2022 a través de una diplomatura integral en Desarrollo Web, Javascript y React Js, lo que me permitió adquirir una sólida base técnica y una visión estratégica del desarrollo digital.

Mi experiencia abarca la realización de proyectos de diversa índole, incluyendo e-commerce, páginas informativas, sitios empresariales y soluciones para emprendimientos. Complemento mi formación académica con una constante actualización profesional, investigando nuevas herramientas y recursos que optimizan la calidad y eficiencia de mi trabajo. Me mantengo al día con las tendencias y avances tecnológicos para ofrecer siempre soluciones innovadoras y de alto estándar.

Mi compromiso es brindar un servicio funcional y transparente, generando un proceso colaborativo con el cliente para entender sus necesidades y poder plasmarlas en su sitio de manera efectiva.


Mis principales recursos y herramientas de trabajo:

- HTML
- CSS
- Javascript
- React Js
- SaaS
- Bootstrap
- Git / GitHub
- Fontshare / Google Fonts
- Inteligencia Artificial (AI)
- Wordpress / Tiendanube
- Hostinger / Netlify
